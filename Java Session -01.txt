JDK's Components
=======================
Javac		: java compiler which convert java code to byte code.
JVM, JIT	: java virtual machine 
			-memory allocation 
			-deallcoation 
			-GC.
		: JIT : just in time which is interpreter which convert byte code to 		native/machine code/binary code
		   			
JRE		: java runtime environment

Keywords/Indentifier 
=============================
package		: is reference of current folder/pacakge 
public		: is access specifier / can be access from anywhere 
class		: is wrapper of data member and functions 
static		: static member, function can be access without object from anywhere
		: single memory allocation 
	
void		: is no return 
main		: is start function / predefined function 
int		:  is data type for numeric 
String		: is data type for alpha numeric 	
if		: condtional 
else
switch
for
etc.

Syntax
==============
//	: single line comment 

/*
	: multi line comments 	
*/
{}	: body of class, function , loop, condition 
()	: function , condition , loop 
[]	: array : collection of data or values 

;	: termination of statement
